# Battle-Paint

A pragmatism Minecraft datapack for a minigame "Battle Paint"

This datapack does only support Chinese(simplified) now, sotty about that.

You should unzip this datapack to your saves' datapack folder and restart your saves before you enjoy this minigame.

Please use WorldEdit mod to paste the map file "map.schem" to the new dimention called "battlepaint:battlepaint" after you restart your saves. Use **/execute in battlepaint:battlepaint tp @s 0 70 0** to teleport yourself to the minigame dimension, then use **//schem load map** and **//paste -o -a** to load the map.

You can use command or use command blocks to teleport players to the minigame dimension. use **/execute as <player> run function battlepaint:battlepaint**.

You can use command or use command blocks to teleport players back to overworld. please edit the teleport command in **battlepaint:lobby/tplobby**.

Enjoy the game : ).
